Real-Time Wildfire Detection Pipeline

An end-to-end geospatial ML pipeline for near-real-time wildfire monitoring. NASA FIRMS thermal data, Sentinel-2 imagery, NOAA weather feeds, and USGS elevation data ingest into PostGIS. Apache Airflow orchestrates all jobs. A CNN smoke classifier scores imagery and flags high-probability fire events, deployed on GCP with a live Kepler.gl spatial dashboard. MS Data Science Capstone — University of Arizona.

Autonomous Fleet Telemetry Data Platform

A production-scale data platform simulating autonomous vehicle telemetry for fleet analytics. High-frequency vehicle streams — GPS traces, speed, heading, sensor events — ingested via Apache Airflow and processed with Spark into dimensional tables in BigQuery for fleet KPIs, trip metrics, and incident tracking.

Flood Potential Assessment — Santa Cruz River Basin

A spatial ML model predicting flood probability at the parcel level. High-resolution LiDAR DEMs provide elevation, slope, and flow accumulation features, classified with a random forest classifier validated through cross-validation and confusion matrix analysis.

Code

EV Charging Site Suitability — Columbus, OH

A multi-criteria spatial optimization model identifying optimal heavy-duty EV charging hub locations in Columbus. Criteria weighted using Analytic Hierarchy Process (AHP) implemented in Python, making the methodology reproducible and defensible for infrastructure planning stakeholders.

Code

Land Use Change Detection — Central Ohio

A 20-year remote sensing study of urban expansion using Landsat 8/9 multispectral imagery. Raw satellite bands classified using a supervised SVM classifier trained on labeled ground truth points, tracking the pace and pattern of sprawl across Central Ohio.

Code
🌀
Historical Hurricane Impact Analysis

A spatiotemporal analysis of hurricane impacts on Gulf and Atlantic Coast infrastructure using NOAA HURDAT2 storm track data (1980–2023). A Geographically Weighted Regression (GWR) model reveals which community factors most predict recovery outcomes across affected regions.

Code

Turbo Roundabout Driver Behavior Study

Graduate research assessing the feasibility of turbo roundabouts using driver performance factors. Built VISSIM microsimulation models, applied logistic regression and ANOVA in SPSS and Python. Published as MS thesis and presented at Ohio Roundabout Conference and AHFE International Conference 2023.

Safety Corridors Effectiveness — Ohio Case Study

Peer-reviewed study evaluating whether safety corridors effectively mitigate crash frequency on Ohio roads. Spatial analysis of crash data across designated safety corridors, applying statistical modeling to assess the effectiveness of enforcement and signage interventions.

Geospatial analysis pipeline supporting FHWA NEVI EV charging deployment plans across Oklahoma, Tennessee, and Mississippi. Identified optimal deployment corridors through Python-automated gap analysis. Published ArcGIS Online web maps for state DOT clients.
